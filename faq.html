WHAT INSPIRED THIS PROJECT?

When I created Media Masterlist, I got feedback that I should use the IGBD API to pull game covers for entries. I loved the idea, but I wanted to keep Media Masterlist minimalistic and flexible for all forms of media. I realized that this was a cool opportunity to make a new project focused only on video games! I wanted to go wild with the design, which was partially inspired by the fun retro styling of the DO IT Hackathon site.

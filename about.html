ABOUT THIS PROJECT

Welcome to GAMER GRID, the eccentric sister-project of Media Masterlist! Media Masterlist is designed to be a simple yet sleek table for tracking your favourite media: books, music, television shows, and more. I wanted to make a more technical project with a similar concept using an API. Creating a playful game-based project seemed like a fun opportunity to try something new and implement some over-the-top styling!
